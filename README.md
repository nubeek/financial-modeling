# Wefranch — Financial Modeling Landing Page

Prototype landing page for Wefranch's franchise financial modeling product. Plain HTML, CSS, and JavaScript — no build step.

## Run locally

Serve the project over HTTP (ES modules won't work via `file://`):

```bash
npx serve .
```

Then open [http://localhost:3000](http://localhost:3000) (port may vary).

## Pages

- `/` — Landing page
- `/models/` — Income statement table demo
- `/ui/` — Product UI prototype
